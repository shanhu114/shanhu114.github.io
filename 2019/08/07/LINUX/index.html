LINUX常用命令

OS 操作系统 （——没有操作系统叫做裸机 ）
Linux （内核）开源 可以自由传播
Linux主要特征
可移动兼容接口 posix 1.0 多用户 多任务 支持多种平台
linux 版本
centOS （多用于数据库） Redhat （多用于开发） ubuntu kali （针对做网络安全和数据取证和攻防演练） suse
Linux系统组成
Linux 内核 ： 储存管理 进程管理 驱动
shell ： 内核和用户交互接口 命令解释器 编程语言
文件系统： 文件存在
应用程序 包括文本编辑器 编程语言 办公自动化软件 数据库等

##文件系统
MBR 4个主分区，最高15个分区 只支持2TB，不支持扩容
GPT 无分区限制 ，支持2TB以上，支持数据扩容
查看分区情况 fdisk -l 查看现有磁盘分区列表
df-a 全部显示
df -l 只显示本地磁盘
df -h 查看分区情况，是以1024为进制显示磁盘容量
df-H 是以1000进制显示磁盘容量
df-T 是显示磁盘分区类型
df-t 指定类型文件分区
df-x 不显示指定类型文件分区
du -b 以byte显示 du -k 以KB显示 du -m 以MB显示
du -h （1024） -H (1000)
du -s 统计指定的文件夹

##查看CPU信息
/proc/cpuinfo cpu信息
/proc/meninfo 内存信息
uname -a 查看linux全部信息
uname -v 内核的编译版本号
uname -r 内核的信息release （发行号)
uname -n 主机名
uname -r |grep -q ^ 2.6 ; echo $?
Ismod 列出内核的所有模块
tcpdump - i eth0 抓本机网卡包 -h 后跟ip
zenamp — 启动图形化nmap

##ssh服务安装配置
1.检查是否安装ssh服务
rpm -qa |grep ssh
2.没有执行这条命令安装
yum install openssh-server
3.修改配置文件
vi /etc/ssh/sshd_config

#PORT 22 去掉#号

#PermitRootLogin yes | no yes允许root ssh登录，no不允许
4.启动ssh服务
systemctl start sshd.service
5.设置开机自启（可选）
systemctl enable sshd.service
